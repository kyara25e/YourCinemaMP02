
<h1 align="center">🎬 YourCinema – Module 02 Project</h1>

<p align="center">
Full Stack web application developed during the <b>Soy Henry Bootcamp (Module 02)</b>.<br/>
A cinema platform where users can explore movie information and interact with dynamic content.
</p>


<h2 align="center">🧠 Project Overview</h2>

<p>
YourCinema is a <b>full stack web application</b> that allows users to browse and explore movies,
view detailed information, and interact with a dynamic interface.
</p>

<p>
This project focuses on building a solid foundation in full stack development,
including frontend structure, backend logic, and API integration.
</p>


<h2 align="center">⚡ Key Features</h2>

<ul>
  <li>🎥 Movie listing and detailed views</li>
  <li>🔎 Dynamic rendering of movie data</li>
  <li>🌐 API integration</li>
  <li>⚡ Interactive user interface</li>
  <li>📱 Responsive design</li>
</ul>

<h2 align="center">🚀 Getting Started</h2>

<h3 align="center">1. Clone the repository</h3>

<pre>
git clone https://github.com/kyara25e/YourCinemaMP02.git
</pre>

<h3 align="center">2. Install dependencies</h3>

<pre>
cd back
npm install

cd ../front
npm install
</pre>

<h3 align="center">3. Run the project</h3>

<pre>
# backend
npm start

# frontend (if needed)
npm run dev
</pre>
